# Fujian Medical Engineering

Base landing page project built with:

- Vue 3
- Vite
- TypeScript
- Tailwind CSS
- vue-i18n

## Included

- Responsive landing page structure
- Sticky header and mobile navigation
- EN / ID / ZH language switching with localStorage persistence
- Placeholder sections for company overview, services, industries, trust signals, and contact

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

This starter uses placeholder content for contact details and trust metrics so it can be quickly customized for a real company website.
